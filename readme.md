## Permissions


### user


```
dice.use
```

### admin


```
dice.admin
```


## Commands


USER SCOPE


- ```/rollr``` -- roll resistance 
- ```/rolld``` -- roll defence
- ```/rolla``` -- roll attack

ADMIN SCOPE


- ```/rolladd [amount] [player]``` -- add a value to the player's roll
- ```/rollremove [amount] [player]``` -- subtract a value from the player's roll
